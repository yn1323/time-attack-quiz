# デザイン基本方針

## 概要
タイムアタッククイズ大会アプリのデザイン基本方針を定義する。
詳細なデザインガイドライン作成前に、方向性を決定した。

## 検討の経緯

デザイン詳細に入る前に決めるべき項目を洗い出し、順次決定した。

### 検討項目と決定内容

| 検討項目 | 選択肢 | 決定 |
|----------|--------|------|
| 雰囲気 | TVクイズ番組風（派手）/ シンプルモダン / レトロゲーム風 | **TVクイズ番組風** |
| テイスト | ポップ / クール / ゴージャス | **ポップ** |
| テーマカラー | ピンク×シアン / オレンジ×イエロー / パープル×ピンク / ライムグリーン×イエロー | **オレンジ × イエロー** |
| 背景 | ダーク / ライト | **ライト** |
| 効果音 | あり / なし | **なし** |
| アニメーション | 派手め / 控えめ | **派手め** |
| デバイス対応 | PC+スマホ / PC専用 | **PC専用** |
| フォント | 丸ゴシック / 太めゴシック / システムフォント | **丸ゴシック** |
| 正解/不正解色 | 緑/赤 / その他 | **緑 / 赤** |

---

## 決定事項まとめ

### 1. 雰囲気・トーン
- TVクイズ番組風
- ポップで楽しい雰囲気

### 2. カラー

| 用途 | 色 |
|------|-----|
| メインカラー | オレンジ × イエロー |
| 背景 | ライト（白系） |
| 正解表示 | 緑 |
| 不正解表示 | 赤 |

### 3. タイポグラフィ
- フォント：丸ゴシック

### 4. 演出
- 効果音：なし
- アニメーション：派手め（カウントダウン、結果発表など）

### 5. デバイス対応
- PC専用（スマホ非対応）
- 管理画面：プロジェクター投影想定

---

## 今後の課題（詳細ガイドライン作成時に決定）

- カラーコードの具体値
- フォントファミリーの具体名
- フォントサイズ・行間の基準
- コンポーネント定義（ボタン、カード、モーダルなど）
- アニメーションの詳細（種類、秒数など）
- レイアウト・グリッド設計
